## Factory Model Explanation

We are a **website factory**, not an agency. This means:

- **Standardized modules** - No custom design work
- **Fast turnaround** - 3-7 days vs 6-12 weeks
- **Predictable pricing** - Clear package pricing, no hourly billing
- **Scalable system** - Can handle high volume
- **Bot-handled config changes** - Small changes via chat, not meetings

## What's NOT Included

- Custom design work outside our module system
- Unlimited revisions (small config changes are allowed; major changes require new factory run)
- Custom development
- Third-party integrations (can be added as add-ons)
- Content creation beyond initial copywriting
