# Client Onboarding Checklist

## Factory Run Process

This checklist tracks a website factory run from intake to launch. Frame client onboarding as "Queue a factory run" rather than "Start a web design project."

## After Intake Submission

### 1. Intake Received ✅
- [ ] Intake form submitted
- [ ] Intake JSON saved to `/leads/{leadId}/intake.json`
- [ ] Validation passed
- [ ] Confirmation email sent (future)

### 2. Proposal Generated ✅
- [ ] Proposal generated from intake
- [ ] Proposal saved to `/leads/{leadId}/proposal.md`
- [ ] Proposal sent to client (future)

### 3. Site Preview Built ✅
- [ ] Site config generated
- [ ] Preview site created at `/app/(examples)/{slug}/`
- [ ] Preview URL shared with client (future)

### 4. Client Review
- [ ] Client reviews preview site
- [ ] Client requests changes (if any)
- [ ] Changes handled via chat bot or manual update
- [ ] Client approves final version

### 5. Agreement & Payment
- [ ] Digital agreement sent
- [ ] Agreement signed
- [ ] Deposit received (50%)
- [ ] Production begins

### 6. Production
- [ ] Final site built from approved preview
- [ ] SEO configured
- [ ] Forms tested
- [ ] Mobile responsiveness verified
- [ ] Final review completed

### 7. Launch
- [ ] Domain configured (if provided)
- [ ] DNS pointed to hosting
- [ ] Site deployed to production
- [ ] Final payment received (50%)
- [ ] Launch confirmation sent

### 8. Post-Launch
- [ ] Client training (if needed)
- [ ] Support plan offered
- [ ] Analytics access provided
- [ ] Follow-up scheduled (30 days)

---

## What Happens After Intake

1. **Immediate (Automated):**
   - Intake validated
   - Proposal generated
   - Preview site created

2. **Within 24 Hours:**
   - Proposal sent to client
   - Preview URL shared
   - Initial contact made

3. **Within 3-7 Days:**
   - Client reviews and approves
   - Agreement signed
   - Production begins

4. **Launch:**
   - Site goes live
   - Client notified
   - Support plan activated (if chosen)

---

## Client Communication

**Initial Contact:**
"Thanks for your intake! We've generated a preview site and proposal. [Link to preview] [Link to proposal]"

**After Approval:**
"Great! We're starting production. Your site will be ready in [timeline]. We'll send you updates as we progress."

**At Launch:**
"Your site is live! [URL] Here's your login info and next steps..."
