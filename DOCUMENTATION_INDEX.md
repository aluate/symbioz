## üìù Documentation Standards

### Naming Conventions
- `CONTROL.md` - Project control documents
- `*STATUS.md` - Status tracking
- `*SUMMARY.md` - Completion summaries
- `README.md` - Project overviews

### File Organization
- Control docs: Project root or `control/` directory
- Status files: Project root
- Guides: Project root or `docs/` directory
- Utilities: `tools/` directory

### Documentation Lifecycle
- **Active:** Current, maintained documentation
- **Archive:** Move to `docs/archive/` when superseded
- **Delete:** Only if truly redundant (after archiving)
