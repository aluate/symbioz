# Sugar Mountain Builders — Operations Playbook

## Overview

This playbook outlines the standard operating procedures for Sugar Mountain Builders, from inquiry to project completion.

---

## Client Pipeline

### 1. Inquiry & Qualification

**Channels**:
- Website form
- Phone
- Email
- Referral
- In-person

**Initial Response**:
- Acknowledge within 24 hours
- Quick qualification (budget, timeline, location, project type)
- Schedule consultation call

**Qualification Questions**:
- Project type (custom, spec, modular, remodel)
- Budget range
- Timeline
- Location/site
- Current status (have land, have plans, etc.)

### 2. Consultation Call

**Process**:
- Understand project goals
- Discuss budget range
- Explain SMB process
- Determine next steps
- Schedule site visit (if needed)

**Documentation**:
- Notes from call
- Project summary
- Next steps
- Follow-up tasks

### 3. Site Visit (if applicable)

**Process**:
- Assess site conditions
- Evaluate access
- Evaluate utilities
- Discuss feasibility
- Refine scope and budget
- Build relationship

**Documentation**:
- Site photos
- Site notes
- Access notes
- Utility notes
- Feasibility assessment

### 4. Proposal Development

**Process**:
- Develop rough budget or detailed estimate (depending on phase)
- Create timeline
- Outline process
- Prepare proposal

**Proposal Contents**:
- Project summary
- Scope of work
- Budget (rough or detailed)
- Timeline
- Process overview
- Next steps

### 5. Proposal Presentation

**Process**:
- Present proposal to client
- Answer questions
- Refine if needed
- Move toward contract

**Follow-Up**:
- Answer questions
- Refine proposal
- Negotiate terms
- Finalize contract

### 6. Contract Execution

**Process**:
- Finalize contract
- Execute agreement
- Collect deposit
- Begin preconstruction

## Construction Phase

### 8. Construction Management

**Process**:
- Manage construction schedule
- Coordinate subcontractors

- Problem-solving

**Weekly Updates**:
- Progress photos
- Schedule updates
- Budget updates
- Issue resolution
- Next week plan

**Client Communication**:
- Regular updates (weekly or as needed)
- Responsive to questions
- Transparent about issues
- Proactive problem-solving

### 9. Modular Install Process (Stax)

**If Stax Module Install**:

**Pre-Install**:
- Coordinate with Stax (production schedule)
- Prepare foundation
- Prepare utilities (stubbed)
- Prepare site access
- Coordinate transport
- Coordinate crane

**Install Day**:
- Coordinate module delivery
- Coordinate crane
- Set modules on foundation
- Connect modules (if multi-module)
- Verify set
- Weather-tight completion

**Post-Install**:
- Final connections (utilities, MEP)
- Exterior finish (decks, steps, porches)
- Interior finish (trim, built-ins)
- Final inspections
- Completion
