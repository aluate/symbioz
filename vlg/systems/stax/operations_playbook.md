## Overview

This playbook outlines the standard operating procedures for Stax modular production, from order to delivery and set.

---

## Order Processing

### 1. Order Receipt

**Channels**:
- Website inquiry (homeowner)
- Builder inquiry
- Developer inquiry
- SMB referral

**Information Required**:
- Module type (Scout, Ridge, Summit, etc.)
- Configuration (single, multi-module)
- Options package (Quartz, Garnet, Sapphire)
- Site location
- Timeline requirements

**Initial Response**:
- Acknowledge receipt (within 24 hours)
- Review for completeness
- Schedule site assessment (if needed)
- Begin quote process

### 2. Site Assessment

**Process**:
- Evaluate site access
- Evaluate foundation requirements
- Evaluate utility availability
- Evaluate transport route
- Evaluate crane access

**Documentation**:
- Site photos
- Access notes
- Foundation requirements
- Utility requirements
- Transport route notes

### 3. Quote Process

**Steps**:
1. Determine module type and configuration
2. Select options package
3. Calculate module pricing
4. Calculate transport cost (based on location)
5. Calculate set cost (if included, or via GC)
6. Estimate timeline

**Quote Delivery**:
- Module pricing
- Options package pricing
- Transport cost
- Set cost (if included)
- Total project cost
- Timeline estimate
- Payment terms

### 4. Order Confirmation

**Requirements**:
- Buyer approval of quote
- Purchase order or signed agreement
- Deposit payment
- Final specifications confirmed
- Site preparation confirmed (foundation, utilities)

**Order Entry**:
- Enter into production queue
- Assign production slot
- Confirm material availability
- Schedule production start
- Coordinate with GC (SMB or partner)

---

## Production Process

### 5. Material Procurement

**Process**:
- Order materials based on module specifications
- Coordinate with suppliers
- Verify material availability
- Schedule material delivery

**Material Categories**:
- Structural (lumber, sheathing, etc.)
- Exterior (siding, roofing, windows, doors)
- Interior (Grid cabinets, flooring, drywall, paint, trim)
- MEP (electrical, plumbing, HVAC rough-in)
- Appliances (if included in options package)

**Grid Cabinet Integration**:
- Order Grid cabinets (internal transfer pricing)
- Coordinate Grid production with Stax production
- Verify Grid cabinet specifications
- Schedule Grid cabinet delivery

### 6. Floor Deck Assembly

**Process**:
- Build floor deck structure
- Install sheathing
- Install insulation (if applicable)
- Install subfloor
- Quality check

**Quality Control**:
- Check dimensions
- Check square
- Check level
- Check for defects

### 7. Wall Panel Assembly

**Process**:
- Build wall panels
- Install sheathing
- Install insulation
- Install windows (if applicable)
- Install doors (if applicable)
- Quality check

**Quality Control**:
- Check dimensions
- Check square
- Check plumb
- Check window/door installation
- Check for defects

### 8. Roof Assembly

**Process**:
- Build roof structure
- Install roofing system
- Install insulation
- Install weatherproofing
- Quality check

**Quality Control**:
- Check dimensions
- Check slope (if any)
- Check weatherproofing
- Check for defects

### 9. MEP Rough-In

**Process**:
- Electrical rough-in (outlets, switches, lighting)
- Plumbing rough-in (water, waste)
- HVAC rough-in (if included)
- Quality check

**Quality Control**:
- Check code compliance
- Check installation quality
- Check for defects

### 10. Interior Finish

**Process**:
- Install Grid cabinets
- Install flooring
- Install drywall
- Paint
- Install trim
- Install fixtures (if included)
- Install appliances (if included)
- Quality check

**Quality Control**:
- Check installation quality
- Check finish quality
- Check for defects
- Final inspection

### 11. Final QC

**Process**:
- Complete final inspection
- Check all components
- Verify specifications
- Check for defects
- Document completion
- Prepare for transport

**Quality Standards**:
- Dimensions within tolerance
- Square and plumb
- MEP code compliant
- Finish quality meets standards
- No defects
- Ready for transport

### 12. Transport Preparation

**Process**:
- Protect module for transport
- Secure components
- Label module
- Document condition
- Coordinate transport
- Coordinate crane (if needed)

**Transport Requirements**:
- Oversize load permits
- Route planning
- Escort vehicles (if required)
- Insurance during transport
