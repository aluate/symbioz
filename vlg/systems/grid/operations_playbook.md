# Grid Cabinet Systems — Operations Playbook

## Overview

This playbook outlines the standard operating procedures for Grid Cabinet Systems production, from order to delivery.

---

## Order Processing

### 1. Order Receipt

**Channels**:
- Website inquiry
- Phone/email
- In-person

**Information Required**:
- Layout or plan
- SKU list (if known)
- Color selection
- Timeline requirements
- Delivery location

**Initial Response**:
- Acknowledge receipt (within 24 hours)
- Review for completeness
- Identify QuickBuild™ eligibility
- Schedule quote (if needed)

### 2. Quote Process

**Steps**:
1. Review layout/plan
2. Translate to SKUs (if needed)
3. Calculate quantities
4. Determine pricing
5. Estimate lead time
6. Identify QuickBuild™ option (if eligible)

**Quote Delivery**:
- SKU list and quantities
- Pricing breakdown
- Lead time estimate
- QuickBuild™ option (if applicable)
- Payment terms
- Validity period (typically 30 days)

### 3. Order Confirmation

**Requirements**:
- Buyer approval of quote
- Purchase order or signed agreement
- Deposit payment
- Final specifications confirmed

**Order Entry**:
- Enter into production queue
- Assign production slot
- Confirm material availability (SSR check)
- Schedule production start

---

## Production Process

### 4. Material Preparation

**SSR Check**:
- Verify SSR inventory levels
- Confirm material availability
- Trigger re-order if needed (for future orders)

**Material Pull**:
- Pull materials from SSR or order from supplier
- Verify quantities
- Verify colors/specifications
- Stage materials for production

### 5. Cutlist Generation

**Process**:
- Generate cutlist from SKU list
- Optimize material usage
- Minimize waste
- Verify dimensions

**CNC Programming**:
- Program CNC router
- Verify tooling
- Set up for production run

### 6. CNC Cutting

**Process**:
- Load material onto CNC
- Run cutlist program
- Verify cuts
- Quality check (dimensions, square)

**Quality Control**:
- Check dimensions
- Check square
- Check for defects
- Mark pieces for next step

### 7. Edgebanding

**Process**:
- Apply edgebanding to cut pieces
- Match colors (5-color core line)
- Verify adhesion
- Trim excess

**Quality Control**:
- Check edgebanding adhesion
- Check color match
- Check trim quality
- Mark pieces for next step

### 8. Drilling & Boring

**Process**:
- Drill/bore holes for hardware
- Hinge holes
- Slide holes
- Assembly holes

**Quality Control**:
- Verify hole locations
- Verify hole sizes
- Check for defects

### 9. Assembly

**Process**:
- Assemble cabinets
- Install hardware (hinges, slides)
- Install handles
- Install soft-close mechanisms
- Verify operation

**Quality Control**:
- Check assembly (square, tight)
- Check hardware operation
- Check door/drawer alignment
- Check soft-close function

### 10. Final QC

**Process**:
- Final inspection
- Check all components
- Verify specifications
- Check for defects
- Clean/prep for delivery

**Quality Standards**:
- Dimensions within tolerance
- Square and plumb
- Hardware functions properly
- No defects
- Clean and ready for delivery

### 11. Pack & Ship

**Process**:
- Package for pickup or delivery
- Label (if needed)
- Document contents
- Coordinate pickup/delivery

**Delivery Options**:
- Customer pickup
- Grid delivery (if available)
- Shipping (if remote)

## QuickBuild™ Process

### QuickBuild™ Flow

**Eligibility Check**:
- Standard SKUs only
- 5 core colors only
- Standard hardware only
- Sufficient lead time (4–7 days)

**Production**:
- Dedicated QuickBuild slots
- SSR materials (no waiting)
- Standardized process
- Fast turnaround

**Quality**:
- Same quality standards
- Same QC process
- Faster delivery
