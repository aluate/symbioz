# Stax — Options Packages

## Overview

Stax modules are offered with three options packages that align with Sugar Mountain Builders' spec home tiers: **Quartz**, **Garnet**, and **Sapphire**.

These packages determine the level of finishes, fixtures, appliances, and interior components included in each module.

---

## Package Philosophy

### Integration with Sugar Mountain

**Shared Spec Tiers**:
- Stax modules use same tier names as SMB spec homes
- Creates consistency across ecosystem
- Makes it easy for SMB to present modular option
- Simplifies decision-making for customers

**Value Proposition**:
- Clear upgrade path (Quartz → Garnet → Sapphire)
- Predictable pricing
- Consistent quality across tiers
- Builder-friendly (easy to specify)

---

## Quartz Tier (Base)

### Positioning

**Target Market**: Budget-conscious buyers, investors, rental properties, basic needs

**Value Proposition**: Modern modular home at entry-level pricing

### Included Components

**Interior Finishes**:
- Standard paint (white or neutral)
- Standard flooring (vinyl plank or similar)
- Standard trim (painted, basic profile)
- Standard doors (hollow core, basic)

**Grid Cabinets**:
- 5-color core line (basic colors)
- Standard SKUs (no custom modifications)
- Standard hardware (2 handle styles)
- Standard soft-close hardware

**Fixtures**:
- Standard bathroom fixtures (basic)
- Standard kitchen faucet (basic)
- Standard lighting (basic fixtures)
- Standard plumbing fixtures

**Appliances** (if included):
- Basic range/oven (if included)
- Basic refrigerator (if included)
- Basic dishwasher (if included)
- No washer/dryer (typically)

**Windows & Doors**:
- Standard windows (energy-efficient, basic)
- Standard entry door (basic)
- Standard interior doors (hollow core)

**HVAC** (if included):
- Basic HVAC rough-in
- Final connections by GC

### Exclusions

- Premium finishes
- Upgraded fixtures
- Premium appliances
- Custom colors or materials
- High-end features

---

## Garnet Tier (Mid)

### Positioning

**Target Market**: Most homeowners, primary residences, quality-focused buyers

**Value Proposition**: Upgraded finishes and features at mid-level pricing

### Included Components

**Interior Finishes**:
- Upgraded paint (premium colors, better quality)
- Upgraded flooring (better quality vinyl, or engineered hardwood)
- Upgraded trim (better profile, premium paint)
- Upgraded doors (solid core, better quality)

**Grid Cabinets**:
- Upgraded colors (premium colors from Grid catalog)
- Standard SKUs (may include some premium options)
- Standard hardware (2 handle styles, may include premium finish)
- Standard soft-close hardware

**Fixtures**:
- Upgraded bathroom fixtures (mid-level)
- Upgraded kitchen faucet (mid-level)
- Upgraded lighting (better fixtures)
- Upgraded plumbing fixtures

**Appliances** (if included):
- Mid-level range/oven (if included)
- Mid-level refrigerator (if included)
- Mid-level dishwasher (if included)
- Washer/dryer hookups (if applicable)

**Windows & Doors**:
- Upgraded windows (better quality, may include larger sizes)
- Upgraded entry door (better quality, may include glass)
- Upgraded interior doors (solid core, better quality)

**HVAC** (if included):
- Upgraded HVAC rough-in
- May include basic HVAC equipment
- Final connections by GC

### Upgrades Over Quartz

- Better finishes throughout
- Upgraded fixtures and appliances
- Better quality materials
- More design options

---

**Fixtures**:
- Premium bathroom fixtures (high-end)
- Premium kitchen faucet (high-end)
- Premium lighting (designer fixtures)
- Premium plumbing fixtures

**Appliances** (if included):
- Premium range/oven (if included)
- Premium refrigerator (if included)
- Premium dishwasher (if included)
- Premium washer/dryer (if applicable)

**Windows & Doors**:
- Premium windows (high-end, large sizes, premium features)
- Premium entry door (high-end, custom, premium features)
- Premium interior doors (solid wood, custom, premium quality)

**HVAC** (if included):
- Premium HVAC system
- May include advanced features (smart thermostats, etc.)
- Final connections by GC

### Upgrades Over Garnet

- Premium finishes throughout
- High-end fixtures and appliances
- Premium quality materials
- Custom options available
- Designer-level features

## Selection Process

### For Homeowners

1. Choose Stax module (Scout, Ridge, Summit, etc.)
2. Select options package (Quartz, Garnet, Sapphire)
3. Review included components
4. Add any additional options (if available)
5. Finalize pricing

### For Builders

1. Specify module type
2. Specify options package (or let customer choose)
3. Standardize on package for production efficiency
4. Use package as sales tool (easy to present tiers)

## Customization Within Packages

### Limited Customization

**Within Package**:
- Color selections (from package-appropriate options)
- Fixture selections (from package-appropriate options)
- Appliance selections (from package-appropriate options)
- Minor layout adjustments (if feasible)

**Package Upgrades**:
- Can upgrade individual components (may incur additional cost)
- Can mix packages (e.g., Sapphire kitchen, Garnet bathrooms)
- Custom quote required for mixed packages

## Integration with Sugar Mountain

### Spec Home Alignment

**SMB Spec Homes**:
- Use same tier names (Quartz, Garnet, Sapphire)
- Same quality levels
- Easy to present modular option alongside site-built
- Consistent customer experience

**Sales Strategy**:
- SMB presents both site-built and modular options
- Same tier names = easy comparison
- Modular may be faster, site-built may be more customizable
- Customer chooses based on priorities
