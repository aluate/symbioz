# Grid Cabinet Systems — SKU Catalog

**Note**: This is a framework for the SKU catalog. Actual dimensions, codes, and specifications should be finalized based on manufacturing capabilities and market requirements.

---

## SKU Structure

**Format**: `[TYPE]-[WIDTH]-[HEIGHT]-[DEPTH]-[OPTIONS]`

**Example**: `BASE-36-34.5-24-STD`

---

## Base Cabinets

### Standard Base Cabinets

**SKU Code**: `BASE-[WIDTH]-34.5-24-[OPTIONS]`

**Standard Widths** (inches):
- 12, 15, 18, 21, 24, 27, 30, 33, 36, 39, 42, 45, 48

**Standard Height**: 34.5 inches

**Standard Depth**: 24 inches

**Options**:
- `STD` — Standard (single door or drawer front)
- `DBL` — Double door
- `DRW` — Drawer base
- `SINK` — Sink base (no center partition)

**QuickBuild Eligible**: Yes (all standard widths, STD/DBL/DRW options)

**Notes**:
- Frameless Euro construction
- Soft-close hardware standard
- Adjustable legs
- Standard toe kick

---

## Closet Systems

### Closet Components

**SKU Code**: `CLOS-[TYPE]-[SIZE]`

**Types**:
- `SHELF` — Adjustable shelf
- `ROD` — Hanging rod
- `TOWER` — Tall tower (uses TALL cabinet structure)
- `BASE` — Base cabinet (uses BASE cabinet structure)

**QuickBuild Eligible**: Yes (standard components)

---

## Garage & Shop

### Garage Cabinets

**SKU Code**: `GAR-[WIDTH]-[HEIGHT]-[DEPTH]-[OPTIONS]`

**Standard Widths** (inches):
- 24, 30, 36, 48

**Standard Heights** (inches):
- 30, 36, 42, 84 (tall)

**Standard Depths** (inches):
- 18, 24

**Options**:
- `BASE` — Base cabinet
- `WALL` — Wall cabinet
- `TALL` — Tall cabinet

**QuickBuild Eligible**: Limited (standard sizes only)

**Notes**:
- Heavier-duty construction
- May use different hardware (non-soft-close acceptable)

## Hardware Options

### Handles

**SKU Code**: `HDL-[STYLE]`

**Styles**:
- `HDL-1` — Handle Style 1 (standard)
- `HDL-2` — Handle Style 2 (alternative)

**QuickBuild Eligible**: Yes (both styles)

**Notes**:
- Two handle SKUs only (as per brand brief)
- Standard finish options (to be defined)

### Hinges & Slides

**Standard Hardware**:
- Soft-close hinges (standard)
- Soft-close slides (standard)
- Blum or equivalent quality

**QuickBuild Eligible**: Yes (standard hardware only)

## QuickBuild™ Eligibility Summary

**QuickBuild Eligible**:
- Standard base cabinets (all widths, STD/DBL/DRW options)
- Standard wall cabinets (standard widths, 30"/36" heights, STD/DBL options)
- Standard tall cabinets (standard widths, PAN option)
- Standard vanity cabinets (standard widths, 30"/32" heights, SING/DBL options)
- Standard closet components
- 5 core colors only
- 2 handle styles
- Standard hardware only

**Not QuickBuild Eligible**:
- Custom sizes
- Custom modifications
- Non-core colors
- Special hardware
- Garage/shop (limited)
