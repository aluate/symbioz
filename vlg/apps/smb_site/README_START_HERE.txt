========================================
  SMB SITE - QUICK START
========================================

TO START THE SITE LOCALLY:
---------------------------

Double-click on: START_LOCAL.bat

That's it! The site will:
- Install dependencies if needed
- Start the server
- Open in your browser

The site will be at: http://localhost:3000

========================================

TO TEST THE FLOOR PLAN BUILDER:
--------------------------------

1. The site opens automatically
2. Click "Floor Plans" in the navigation
3. Scroll down to see the builder
4. Load a template (Sugarline 65, etc.)
5. Drag rooms from the library to the canvas!

========================================

TO STOP THE SERVER:
-------------------

Press Ctrl+C in the window that opens

========================================

IF IT DOESN'T WORK:
-------------------

1. Make sure Node.js is installed
   Download from: https://nodejs.org/
   (Get the LTS version)

2. If you see errors, check the window
   that opens - it will show what went wrong
