# Life OS Frontend

Next.js 14 frontend for Life OS household management app.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Structure

- `app/` - Next.js app router pages
- `components/` - React components
- `lib/` - Utilities and API client

## Features (Planned)

- Kanban board for tasks
- Dashboard overview
- Calendar view
- Financial tracking
- Admin panel
