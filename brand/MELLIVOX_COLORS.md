# Mellivox Color Palette

Quick reference for the Mellivox color system.

## CSS Variables

```css
:root {
  /* Primary */
  --mellivox-hive-black: #0B0B0F;
  --mellivox-vox-gold: #C69A3E;
  
  /* Secondary */
  --mellivox-pollen-ivory: #EDE8D1;
  --mellivox-forage-green: #2A4E39;
  --mellivox-resin-amber: #B86022;
  
  /* Accent */
  --mellivox-obsidian-violet: #3A2F53;
}
```

## Usage Guidelines

- **Hive Black + Vox Gold:** Sacred/hero moments (logos, seals, key UI screens)
- **Pollen Ivory:** Text backgrounds, parchment, soft contrast
- **Forage Green:** Ecological accents, nature themes
- **Resin Amber:** Ritual highlights (use sparingly)
- **Obsidian Violet:** Rare mystical/glitch accents
