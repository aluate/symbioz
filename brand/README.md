# Mellivox Brand Documentation

This directory contains the complete brand identity documentation for **Mellivox**, the overarching brand and doctrine for the Symbioz universe.

## Files

- **`MELLIVOX_BRAND_BIBLE.md`** - Complete brand identity, including:
  - Core identity and brand pillars
  - Symbol & logo philosophy
  - Color system
  - Voice & tone guidelines
  - Lore hooks (non-mechanic)
  - Gameplay boundaries
  - Key phrases & mantras
  - **DAC (Devil's Advocate Check)** section with intentional brand edges

- **`MELLIVOX_COLORS.md`** - Quick reference for the color palette with CSS variables and Tailwind config examples

- **`MELLIVOX_LOGO_SPEC.md`** - Logo and design specifications, including:
  - Design requirements for the bee sigil
  - Logo lockup specifications
  - File structure recommendations
  - Usage examples
  - DAC notes specific to logo design

## Quick Reference

**Core Theme:** Symbiosis produces consequence. Every connection changes reality.

**Mantra:** "The Hive remembers."

**Colors:**
- Primary: Hive Black (`#0B0B0F`), Vox Gold (`#C69A3E`)
- Secondary: Pollen Ivory, Forage Green, Resin Amber
- Accent: Obsidian Violet

**Key Principle:** Mellivox influences *aesthetic, mood, lore flavor*, not gameplay mechanics.
